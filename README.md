## 개요
Next.js 13 학습을 위해 패스트캠퍼스 인터넷 강의를 수강하며 만든 퀴즈 앱입니다.<br/>
개발에 Next.js 13, tailwind가 사용되었고 배포에는 vercel을 이용했습니다.
#### 📝<a href="https://next-quiz-app-weld.vercel.app/">**퀴즈 앱 바로가기**</a>
